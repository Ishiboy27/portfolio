Fire Service Coverage Study

Utilized ArcGIS Pro's Network Analysis tools, including Service Areas and Location Allocation, to determine current fire service coverage within a 3.5-minute response time from existing fire stations in London, Ontario. Also calculated Census data for all the generated polygons.

Conducted two key scenario analyses:

  • Scenario 1: Explored optimal locations to add a new fire station by analyzing building data and identifying areas with the highest concentration of uncovered buildings, ensuring maximum coverage improvement and calculating the improved census data.
  • Scenario 2: Simulated the removal of an existing fire station to assess the impact on service coverage, selecting the option that resulted in minimal service loss while maintaining response time efficiency and calculating the subsequent census data.

Identification of Relative Vegetative Biomass in Westmount and Byron Districts

This project utilized remote sensing techniques and GIS analysis to assess the relative vegetative biomass in the Westmount and Byron districts of London, Ontario. The study focused on using different raster imagery, including RGB, infrared, and NDVI (Normalized Difference Vegetation Index), to distinguish vegetative cover from urban areas.

Key Imagery Used:

  • RGB Image: Showed natural color and vegetation distribution across the districts.
  • Infrared Image: Differentiated vegetation from non-vegetated areas using false color.
  • NDVI: Quantified vegetative biomass, highlighting green areas based on vegetation health.

Both districts displayed substantial vegetation cover, particularly near the Thames River, making them valuable zones for urban green space management.

Refining Image Classification in Westmount Planning District

This project focused on refining the supervised classification results for perviousness in a selected planning district of London. The initial classification encountered issues like misclassified shaded areas and overlapping tree canopy. To resolve these, Boolean raster data layers for impervious features (e.g., buildings, roads) and pervious features (e.g., vegetation, water) were created. These layers were integrated with the original classification using the Raster Calculator tool in ArcGIS Pro to produce an improved classification.

The final map layout compares the original and refined classifications, providing a quantitative summary of area-based improvements, aiding better land-use planning and analysis. The results showed the following changes:

  • Increase in accurately classified pervious areas by 15%.
  • Reduction in misclassified impervious areas by 10%.
  • Significant improvement in distinguishing shaded regions and tree canopy overlaps.
